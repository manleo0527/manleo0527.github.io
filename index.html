一、   注册查询类接口

接口说明：

1）所有系统错误响应格式均为

{

error: {

code: -23655,

message: '注册失败'

}

}

2）请求地址格式：

http://子系统名.coolact.mchuan/api/XML名称?{method:’方法名’,params:{}}

例如：

http://activity.coolact.mchuan/api/plugin?{method:’regist’,params:{}}

3）分页查询多条数据响应格式：

{

totalcount: 总数,

pagenumber: 页数,

pagesize: 每页显示条数,

rows: []  //具体数据

}

6.       插件页面修改接口

  • 请求链接: 
  • http://localhost:8080/coolact/japi/page?{method:'update',params:{}}
